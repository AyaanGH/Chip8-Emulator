# Chip8-Emulator

![Pong emulated on a Chip 8 system!](pong.png)




*This projects is an emulator designed to run the 1970's Chip-8 system*



## Specs
- 34 instructions

- 4 kb of  memory

- 16 8 bit registers

- A stack of 16 16-bit values

- Stack pointer

- 64 x 32 pixels display

- 16 keys
